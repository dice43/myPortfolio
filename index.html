Projects

Auto Cue

Oct 2024 - Oct 2024

A python script used to automate the setting of cue points in Serato DJ software. This code is based off of Bide's Rekordbox AutoCuer v1.0, and was reconfigured to work on Windows. The program is customizable and user can set parameters for cue point distances.

Leaf Highlight

Sept 2023 - Sept 2023

A program that takes android screenshots and xml metadata dumps from android's UIAutomator Framework and uses them to highlight the leaf-level GUI components that are present in the xml files. The program outputs screenshots with these leaf components highlighted by yellow rectangles.

Internet Cookbook

May 2022 - Jul 2022

A website that interacts with a recipe API to generate recipes based on a search. Users can search for recipes to save to their account for viewing later. Account information is obfuscated using salt and hashing to protect private information. Support for free heroku is not available anymore so the project was abandoned.

Record Receipts

Sep 2022 - Oct 2022

An app that can be used to track a user's budget across many different categories. The user may take a photo of their receipt with the camera and save an image of it to the app's persistent internal storage. The image is then processed with machine learning Optical Character Recognition. Application was made with a focus on application security and solid functionality.
